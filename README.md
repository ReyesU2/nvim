
## How to add this config to neovim:

### Linux and Mac
      git clone https://github.com/ReyesU2/nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim

### Windows
  If you're using cmd.exe:
    
    git clone https://github.com/ReyesU2/nvim.git "%localappdata%\nvim"
  If you're using powershell.exe

    git clone https://github.com/ReyesU2/nvim.git "${env:LOCALAPPDATA}\nvim"
